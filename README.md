# Romania Site — IT Certificate Project

A **static website about Romania** built with HTML, CSS, and JavaScript, created as the practical project for the **Romanian IT Digital Competences Certificate** (Atestat de competențe digitale).

## Overview

This website was developed by Popescu Constantin Mădălin as a high school graduation project (lucrare de atestat) for the IT certification exam. It presents information about Romania — history, geography, culture, and tourism — in a multi-page static web format.

## Features

- Multi-page static website
- - Informational content about Romania
  - - Styled with CSS
    - - JavaScript interactivity
      - - Built entirely with vanilla HTML/CSS/JS — no frameworks
